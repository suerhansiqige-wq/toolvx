Photo Anonymization for Journalists

Protect your sources and maintain journalistic integrity

Why Source Protection is Critical

For investigative journalists, protecting source identities isn't just ethical—it can be life-saving. Whether you're working with whistleblowers, protest participants, or victims of crime, properly anonymizing photos before publication is essential for maintaining trust and ensuring safety.

Key Principles:

  • • Always obtain informed consent when possible
  • • Use multiple anonymization layers
  • • Remove ALL metadata from images
  • • Test images with facial recognition software
  • • Consider context clues (tattoos, clothing, locations)

Complete Anonymization Checklist

1. Facial Features

  • • Blur entire face including hair line
  • • Cover distinctive features (scars, tattoos)
  • • Obscure eye shape and color
  • • Use heavy pixelation or solid masks

2. Identifying Marks

  • • Cover unique jewelry or accessories
  • • Blur visible tattoos
  • • Hide birthmarks or scars
  • • Obscure distinctive clothing/logos

3. Background Elements

  • • Blur recognizable locations
  • • Remove street signs or landmarks
  • • Obscure license plates
  • • Hide building numbers

4. Metadata Removal

  • • Strip EXIF data (camera info, GPS)
  • • Remove creation date/time
  • • Delete device identifiers
  • • Use tools that don't store metadata
